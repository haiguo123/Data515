\# User Stories

1\. Paul the player

* Want: Casual, for fun, player, want's to test movie knowledge
* Interaction methods: User inputs through web UI provided by creators
* Need: They need a internet connection and computer/smart phone
* Skills: Movie and actor knowledge, ability to use basic/intuitive web interfaces





2\. Candice the competitor

* Want: Test movie knowledge, but on a competitive level, wants to place on the leaderboards
* Interaction methods: User inputs through web UI provided by creators, paying attention to leaderboards as well
* Need: They need a internet connection and computer/smart phone
* Skills: High level of movie and actor knowledge, strong familiarity with the UI



3\. Owen the organizer

* Want: Create custom game(s) for others 
* Interaction methods: User inputs through web UI provided by creators. Different interface from players, where Owen could setup/create a game, with the option to designated start and end points as opposed to randomized ones (starting movie/actor and target movie/actor).
* Need: They need a internet connection and computer/smart phone, optional friends/others as players unless they are simply setting up a game for themself
* Skills: Movie and actor knowledge, ability to organize a game for others.
